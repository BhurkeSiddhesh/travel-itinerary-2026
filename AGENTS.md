# AGENTS.md — Project DNA

## Tech Stack
- HTML5 (Semantic structure)
- Vanilla CSS (Custom tokens, premium aesthetics)
- Vanilla JavaScript (Page navigation, interactivity)
- Google Fonts (Cormorant Garamond, Jost)

## Structural Rules
- Single-page application (SPA) style with hidden/visible sections.
- Sidebar-based navigation.
- Responsive design for desktop and mobile (printing support included).

## Known Patterns and Constraints
- Use rich aesthetics: vibrant colors, glassmorphism, dynamic animations.
- No external CSS frameworks (unless requested).
- High-quality imagery for location visualization.

## Active Focus
- Integrating actual ticket bookings (safaris, accommodations, flights, car rentals) and updating route details to reflect the cancellation of Mysuru in favor of Nagarhole Safari and morning elephant camp.
- Maintaining the premium "Travel Guide" feel across desktop and mobile devices.
